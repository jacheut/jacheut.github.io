好处：

  • Claude 先探索代码库

及早纠错

如果 Claude 开始偏离正确方向：

  • 按 Escape 立即停止
  • 使用 /rewind 或双击 Escape 恢复到之前的检查点
  • 重新调整方向

常用命令

快捷键

快捷键 功能
Enter 发送消息
Shift + Enter 换行（不发送）
Escape 停止当前操作
Double Escape 恢复到上一次检查点
Ctrl + C 退出 Claude Code
Shift + Tab 进入/退出 Plan Mode
